Graphics.lineDraw(x0, y0, x1, y1, color)
Draws a line on LCD

Example

var lcd = new Dogs102x6();
var g = new Graphics(lcd);
g.lineDraw(10, 10, 20, 20, 0xffff);

Params

Parameter name Description
x0

X0 coordinate
y0

Y0 coordinate
x1

X1 coordinate
y1

Y1 coordinate
color

Color

Return Value

None
