Graphics.roundRectDraw(x, y, w, h, r, color)
Draws a round rectangle on LCD

Example

var lcd = new Dogs102x6();
var g = new Graphics(lcd);
g.roundRectDraw(10, 10, 20, 20, 4, 0xffff);

Params

Parameter name Description
x

X coordinate
y

Y coordinate
w

Width
h

Height
r

Corner Radius
color

Color

Return Value

None
