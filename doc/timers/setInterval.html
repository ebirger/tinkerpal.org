Timer.setInterval(cb, ms)
Schedule 'cb' to run periodically every ms milliseconds

Example

setInterval(function() { debug.dump('.'); }, 1000);

Params

Parameter name Description
cb

Callback to be run
ms

Period in milliseconds

Return Value

Timer ID that can be used in clearTimeout()
