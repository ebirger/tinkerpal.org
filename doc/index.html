• tink.js 0.1.0 API
API description

Functions

Global

Function Description
setTimeout Schedule 'cb' to run in ms milliseconds
setInterval Schedule 'cb' to run periodically every ms milliseconds
clearTimeout Cancel timeout timer if timeout hadn't expired
clearInterval Cancel periodic timer
getTime Get number of seconds since system startup
eval Evaluate the given String
toInteger Convert the given object to an integer value

Debug

Function Description
assert Panic on mismatch between two objects
assert_cond Panic if object is false
dump Dump object to the console

Math

Function Description
sin Compute the sine of an angle
asin Compute the arc sine of a number
cos Compute the cosine of an angle
acos Compute the arc cosine of a number
tan Compute the tangent of an angle
atan Compute the arc tangent of a number
sqrt Compute the square root of a number

Array

Function Description
push Add object to tail of array

File System

Function Description
readFileSync Synchronously reads the entire contents of a file
readdirSync Synchronously reads the directory listing of a path

GPIO

Function Description
digitalWrite Set the digital value of a GPIO or a number of GPIOs
digitalPulse Create a digital pulse on a GPIO pin for a given period
digitalRead Read the digital state of a GPIO pin or a number of pins
analogWrite Set the analog value of a GPIO pin
analogRead Read the analog value of a GPIO pin
setWatch Calls a callback function whenever the GPIO pin changes state

Serial

Function Description
enable Enable a serial port
disable Disable a serial port
print Prints string
onData Calls 'cb' when data is available on serial port
setConsole Sets tink.js console to the given serial port

Platform

Function Description
meminfo Prints platform dependent memory information
