• tink.js 0.1.0 API
API description

Functions

Global

Function Description
eval Evaluate the given String
toInteger Convert the given object to an integer value
meminfo Prints platform dependent memory information

Timer

Function Description
setTimeout Schedule 'cb' to run in ms milliseconds
setInterval Schedule 'cb' to run periodically every ms milliseconds
clearTimeout Cancel timeout timer if timeout hadn't expired
clearInterval Cancel periodic timer
getTime Get number of seconds since system startup

debug

Function Description
assert Panic on mismatch between two objects
assert_cond Panic if object is false
dump Dump object to the console
dump_env Dump global environment information to the console

Math

Function Description
sin Compute the sine of an angle
asin Compute the arc sine of a number
cos Compute the cosine of an angle
acos Compute the arc cosine of a number
tan Compute the tangent of an angle
atan Compute the arc tangent of a number
sqrt Compute the square root of a number
log Compute the natural logarithm of a number
exp Compute the base-e exponent of a number
floor Compute the largest integral value not greater than the argument
ceil Compute the smallest integral value not less than the argument
round Round to nearest integer, away from zero
abs Compute the absolute value of an integer
atan2 Compute the arc tangent of two variables
pow Power function

Module

Function Description
require Searches for a module and evaluates its code

Function

Function Description
call Invoke function call

Array

Function Description
push The arguments are appended to the end of the array, in the order in which they appear
pop The last element of the array is removed from the array and returned
forEach Calls cb once for each element present in the array, in ascending order
indexOf Searches for searchElement in the elements of the array
join Joins the string coversions of the elements of the array separated by occurrences of the separator
map Calls cb once for each element present in the array, in ascending order, and constructs a new Array from the results

Object

Function Description
toString The object is converted to a string

String

Function Description
split Breaks a string into substrings based on occurences of 'separator'
indexOf Search for occurences of 'searchString' in a given string
substring Creates a new string based on a subset of a given string
charAt Creates a new string containing the character at a position
charCodeAt Get the ASCII value of the character at a position

File System

Function Description
readFileSync Synchronously reads the entire contents of a file
readdirSync Synchronously reads the directory listing of a path

GPIO

Function Description
digitalWrite Set the digital value of a GPIO or a number of GPIOs
digitalPulse Create a digital pulse on a GPIO pin for a given period
digitalRead Read the digital state of a GPIO pin or a number of pins
analogWrite Set the analog value of a GPIO pin
analogRead Read the analog value of a GPIO pin
setWatch Calls a callback function whenever the GPIO pin changes state

Serial

Function Description
enable Enable a serial port
disable Disable a serial port
print Prints string
onData Calls 'cb' when data is available on serial port
setConsole Sets tink.js console to the given serial port
