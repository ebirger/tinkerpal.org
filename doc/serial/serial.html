• TinkerPal 0.1.0 API /
  • Serial

Serial

>>>

enable
Enable a serial port

Examples

var s = new Serial(UART1);
s.enable();
>>>

disable
Disable a serial port

Examples

var s = new Serial(UART1);
s.disable();
>>>

print
Prints string

Examples

var s = new Serial(UART1);
s.print('Hello World!');

Params

Parameter name Description
string

String to be printed to serial port
>>>

onData
Calls 'cb' when data is available on serial port

Examples

var s = new Serial(UART1);
s.onData(function(e) { s.print(e.data); });

Params

Parameter name Description
cb

callback function accepting a data object
>>>

setConsole
Sets TinkerPal console to the given serial port

Examples

var s = new Serial(UART1);
s.setConsole
