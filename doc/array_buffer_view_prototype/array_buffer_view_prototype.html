subarray
Return a partial typed array based on the typed arrayIf end is unspecified, the subarray contains all elements from begin to the end of the TypedArray. If either begin or end are negative, they are calculated from the end of the array

Examples

var a = new Int8Array(16);
var b = a.subarray(1, 5);

Params

Parameter name Description
begin

Start offset (inclusive)
end

(Optional) End offset (exclusive)
