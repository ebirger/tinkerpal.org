Function

>>>

call
Invoke function call

Examples

function fun() { this.prop = 'prop' }
var obj = {};
fun.call(obj);
obj; // { prop : 'prop' }

Params

Parameter name Description
this

The 'this' value on function invokation
arg1, arg2, ...

[optional] arguments to pass to function
