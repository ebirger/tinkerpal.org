GPIO Functions.digitalRead(pin[s])
Read the digital state of a GPIO pin or a number of pins

Example

var state = digitalRead(GPIO_PF2);
var a = digitalRead([GPIO_PF0, GPIO_PF1, GPIO_PF2

Params

Parameter name Description
pin[s]

GPIO pin ID or array of GPIO pin IDs

Return Value

Single pin - boolean GPIO state Multiple pin array - number representing the pins state where the first pin is the MSB
