digitalPulse(pin, value, ms)
Create a digital pulse on a GPIO pin for a given period

Example

digitalPulse(GPIO_PF2, true, 0.1);

Params

Parameter name Description
pin

GPIO pin ID
value

Boolean
ms

Pulse period in milliseconds

Return Value

None
