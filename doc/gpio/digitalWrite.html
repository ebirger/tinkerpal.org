GPIO Functions.digitalWrite(pin[s], value)
Set the digital value of a GPIO or a number of GPIOs

Example

digitalWrite(GPIO_PF2, true); /* Turn on PF2 */
digitalWrite([GPIO_PF1, GPIO_PF2, GPIO_PF3], 0x5); /* Turn on PF1 & PF3 */

Params

Parameter name Description
pin[s]

Single GPIO ID or array of GPIO IDs
value

Boolean for a single GPIO In case of an array, value is considered an integer where the first array element maps to the MSB

Return Value

None
