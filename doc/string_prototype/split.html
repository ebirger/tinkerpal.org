String.split(separator)
Breaks a string into substrings based on occurences of 'separator'

Example

var s = '1|2|3';
var a = s.split('|');
debug.assert(a, [ '1', '2', '3' ]);

Params

Parameter name Description
separator

Delimiter

Return Value

Array of strings, 'separator' is not part of the strings
