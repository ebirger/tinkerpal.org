Etherif.packetRecv()
Receive packet data

Example

var e = new ENC28J60(SPI1, GPIO_PE3, GPIO_PF4);
e.onPacketReceived(function() { console.log('packet size: ' + e.packetRecv().length); });

Return Value

Uint8 array buffer view with the packet data
