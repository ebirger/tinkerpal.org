    Lightweight JavaScript engine and ecosystem for embedded platforms

    What?

    A partially ECMA-262 compliant JavaScript interpreter, with a very small footprint and a peripheral access library.

    Runs nicely on MCUs and allows easy prototyping

    Why?

    The world really needed another JavaScript engine implementation, really!

    Yeah right. Especially when there are already several for MCUs.

    The thing is, using JavaScript to program MCUs is quite awesome, and doing it on different MCUs, with all the constraints imposed by limited resources, is a fun project to hack.

    Plus, the code base allows a nice starting point for other MCU projects, as it provides utility features such as:

    • Hardware independent peripheral access API
    • Timers
    • Memory management
    • Build/Configuration infrastructure

    Features

    • Somewhat functional
    • Command Line Interface
    • Modules Support
    • Portable C99 Source Code
